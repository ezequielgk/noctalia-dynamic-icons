#  Noctalia Dynamic Icons
Dynamic icon color synchronization for GTK file managers (Nemo, Thunar, Nautilus, Caja). 
Powered by **Noctalia** and Git-based color snapshots.

##  Installation

Choose your preferred icon pack and run the command in your terminal.

### ðŸ”¹ Papirus-Dark
Classic, flat, and highly compatible.
```bash
# Bash / Zsh
curl -fsSL [https://raw.githubusercontent.com/ezequielgk/noctalia-dynamic-icons/main/papirus.sh](https://raw.githubusercontent.com/ezequielgk/noctalia-dynamic-icons/main/papirus.sh) | bash
```

```bash
# Fish Shell
bash (curl -fsSL [https://raw.githubusercontent.com/ezequielgk/noctalia-dynamic-icons/main/papirus.sh](https://raw.githubusercontent.com/ezequielgk/noctalia-dynamic-icons/main/papirus.sh) | psub)

```
